# Municipal Wall Crack Detection

This project detects cracks in municipal walls using image processing techniques.

## 📌 Project Description
Wall cracks in buildings and public infrastructure can indicate structural damage.  
This project uses computer vision techniques to detect cracks from images automatically.

## 🛠 Technologies Used
- Python
- OpenCV
- NumPy
- Jupyter Notebook

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/Goutham91/MUNICIPAL-WALL-CRACK-DETECTION-.git
```

2. Install required libraries

```
pip install opencv-python numpy matplotlib
```

3. Run the script

```
python crack_detection.py
```

Wall images containing cracks.
